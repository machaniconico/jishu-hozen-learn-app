# 自主保全士 学習アプリ

自主保全士（1級・2級）の試験対策学習サイト。テキスト解説・確認クイズ・フラッシュカード・模擬試験を1つのアプリで。

## 試験範囲

公益社団法人 日本プラントメンテナンス協会（JIPM）が実施する自主保全士検定の以下5科目を網羅。

1. **生産の基本** — TPM・5S・安全衛生・QCD・標準作業
2. **生産効率化とロスの構造** — 7大ロス・設備総合効率・OEE
3. **自主保全活動** — 自主保全7ステップ・清掃・点検・給油・増締め
4. **改善・解析の手法** — なぜなぜ分析・PM分析・QC七つ道具
5. **設備保全の基礎** — 機械要素・潤滑・油圧・空気圧・電気・計測

## 学習モード

- 📖 **テキスト学習** — 体系的に各科目を学ぶ
- ✅ **確認クイズ** — レッスンごとの理解度チェック
- 🃏 **フラッシュカード** — 用語暗記用
- 📝 **模擬試験** — 本番形式の過去問風問題
- 📊 **進捗ダッシュボード** — 学習履歴を可視化

## 技術スタック

- Next.js 16 (App Router, Static Export)
- React 19
- TypeScript
- Tailwind CSS 4
- localStorage（進捗保存）
- Cloudflare Pages（デプロイ）

## 開発

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # 静的ビルド → out/
```

## デプロイ

Cloudflare Pages にて `out/` を公開。独自ドメイン: `jishu-hozen-learn.app`
